This is the source code of PhSHI!
Please use jdk-11.0.2 or higher version!
Kafka version is 2.11-1.1.0!

This code is just uesed for test, if the code is applied in other scenarios, problems may occur. Please use with caution!

The Client folder contains the generation, sending, and receiving of subscriptions. The core folder contains static versions of rein, tama and opindex and adaptive versions of rein. The EventSender folder contains the sending of events.
